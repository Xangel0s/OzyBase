A high-performance management interface for OzyBase, built with React, Tailwind CSS, and Lucide React.

## Features
- **Modern UI**: Dark mode, glassmorphism, and neon accents.
- **Table Editor**: Spreadsheet-like experience for managing your data.
- **Responsive**: Works on all screen sizes.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
